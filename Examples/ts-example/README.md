# Intall and Run

Download example project and install node modules
```
git clone https://github.com/Thanasis1101/react-native-gif-search
cd Examples/ts-example
npm install
```

Start node server
```
react-native start
```

Run on device
```
react-native run-android
# or
react-native run-ios
```
